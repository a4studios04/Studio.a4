Getting started

Start animating with our powerful API

Basic Animation

animate('.square', {
  rotate: 90,
  loop: true,
  ease: 'inOutExpo',
});

Random Animation

animate('.shape', {
  x: random(-100, 100),
  y: random(-100, 100),
  rotate: random(-180, 180),
  duration: random(500, 1000),
  composition: 'blend',
});
